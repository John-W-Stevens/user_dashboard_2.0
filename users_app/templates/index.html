User Dashboard

This full-stack application is built with Django 2.2. The following technologies were used in the development of this app: Python3, Django 2.2, Bootstrap 4, HTML, CSS, and JavaScript

Major features include login and registration, account administration, and a message board.

Get Started

Manage Users

With an admin account, you can register new users, remove current users, edit user profiles, and change user permissions.

Leave Messages

Leave messages and comments for each user on their wall. Edit and delete your own messages and comments. Admin's can edit/delete any message or comment

Edit User Information

Users are able to edit their profile information. Administrators can edit the profiles of other users and change their permissions.
